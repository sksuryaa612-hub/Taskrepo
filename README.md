5. ✖️ Multiplication Table Generator  
6. 🔢 Digit Counter  
7. 🔄 Number Reverser  
8. 🧮 Factorial Calculator  
9. 🟢 Prime Number Checker  
10. ⭐ Star Pattern Builder
